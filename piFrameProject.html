RaspberryPi Picture Frame

Have you ever seen those fancy digital picture frame things? Well I decided to make one myself.
Why a Picture Frame?

For my first foray into programming hardware with software, I wanted to create something that I've wanted for a while now: a digital picture frame. The project seemed like a great learning opportunity for practicing programming and assembling physical hardware components. Displaying a picture on hardware for the first time is a major milestone for everyone who decides to tinker with single board computers, one that I needed to fulfill for myself.

Hardware

Components:

  • Raspberry Pi Zero W 2
  • Pimoroni Inky Impression 5.7"
  • 3D Printed Case designed by myself

Upon researching hardware to use for this project, I came across these super cool digital e-ink displays made by pimoroni, which are similar to a kindle, but multicolor. These displays are plug and play with any raspberry pi header, making the hardware a breeze to put together. Another cool fact is that they will keep displaying images without power to the computer, a feature that will come in handy later when I make the device wireless. Putting together the hardware is as simple as plugging in the screen to the raspberry pi header, then installing a microSD card into the raspberry pi with raspberry pi OS installed. To display the project, I made a picture frame in CAD with a slot for the screen and raspberry pi combo, then 3D printed the STL.

Software

Utilized technologies:

  • Python
  • Flask
  • HTML & CSS

One of the big goals of this project was to make it extremely user friendly. I intended to make sure my girlfriend would be able to easily access the Raspberry Pi and upload pictures without my help. Therefore, I figured I'd make a web interface so that anyone on my network can add pictures of their own. To make this web interface I used Flask, a python web application framework. Flask would host a webpage written in HTML, and would take user input in the form of pictures.
