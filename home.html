RaspberryPi eInk Frame

This project utilizes a Raspberry Pi Zero 2 W and a Pimoroni 5.7 inch eInk display housed in a 3D printed stand designed by myself in CAD. Being my first venture into utilizing hardware with code, this was my first venture into programming embedded systems. This project uses open source drivers supplied by Pimoroni, and flask, a python framework for building web applications. I was able to create a simple interface with HTML and CSS that allows users to upload and update the display from an internet browser, as long as the user is connected to the same network as the device.
